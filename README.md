# Localizador de Contatos de Empresas (Python)

Este projeto automatiza a busca e extração de informações de contato (Telefone, E-mail e Endereço) para listas volumosas de empresas do Brasil, utilizando Python e técnicas de Web Scraping.

## 🚀 Como Funciona
O script recebe uma planilha Excel com os nomes das empresas na primeira coluna, realiza consultas automatizadas na web simulando um navegador real para evitar bloqueios, e exporta um novo arquivo Excel estruturado com os dados localizados.
